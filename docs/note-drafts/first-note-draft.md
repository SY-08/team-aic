# [DRAFT] TEAM AIC、はじまりの記録

> この下書きは、GitHub Pagesに蓄積した内容を、タイミングを見てnoteへコピペするための方針に沿って作成しています。
> まだ推敲前のドラフトです。

---

TEAM AIC｛AI Circle｝は、AIを活用しながら教育・福祉・地域産業・政治・経済・暮らしの課題を研究し、
町田を起点に小さく実践しているオープンソースプロジェクトです。

ここでいうオープンソースプロジェクトとは、完成した成果だけを発表するのではなく、
活動の過程・学び・仕組み・ノウハウをできるだけ公開しながら進める取り組みを意味します。

今日、その入り口となる公式サイトをGitHub Pagesで公開しました。
サイトには、活動領域の紹介だけでなく、

- 今日やったこと・決めたこと・次にやることを並べる「Live Build Log」
- 設計図を置く「Blueprints」
- AIへの指示やプロンプトを置く「Prompts」
- 将来の本化素材を置く「Book Materials」
- 撮影・編集をしない動画案を置く「YouTube Drafts」

といった、進行中の記録をそのまま残せる場所を用意しています。

現時点では個人活動として研究・実践を進めていますが、
政治や行政にどう関わるべきか、地域や社会をより良くするには何が必要かについては、可能性を閉じずに考え続けています。

まずは町田から、小さく、しかし記録を残しながら進めていきます。
