# テスト用活動記録：ナレモンページ制作とGit/GitHubの学び

このファイルは教材生成のテスト用に、Notionの公開可能な活動記録を一般化したものです。

## 実際に行った作業

- AIクリエイティブ自由研究ページとナレッジモンスターのページを制作・調整した。
- ロゴ、スタンプ、キャラクター画像、漫画画像をHP用素材として追加した。
- ナレッジモンスターマンガのInstagram導線を追加した。
- GitHubで作業用ブランチを使い、Pull Requestを作成し、内容確認後にMergeした。
- 公開後の見た目を確認し、トップビューとボタン配置を修正した。

## 記録されていた学び

- ブランチは、本番とは別の場所で試す道。
- コミットは、ここまでの変更を記録するセーブポイント。
- pushは、PC側の変更をGitHubへ送ること。
- Pull Requestは、変更を本編へ入れてよいか確認する場。
- Mergeは、確認した変更を本番へ統合すること。
