# 活動共有ノートの自動記録

## 正本

Notionの `08-06 活動共有ノート` 直下にある「活動ログ」データベースを正本にする。

公開サイトには、GitHub Actionsが作る `記録元 = GitHub自動集計` の記録と、Notionで `公開する` をオンにした手動記録だけを表示する。手動メモや未整理の振り返りは、チェックをオンにしない限り公開されない。

## 流れ

1. `main` に変更が入る、または毎日23:55（日本時間）になる
2. GitHub Actionsがその日のコミットを読む
3. 専門用語のやさしい説明とGitHub参照を付け、Notionに1日1本の記録を作成・更新する。`main` 更新時は直近14日分を再集計し、初回の履歴も取り込む。
4. Notion同期が活動ログを読み、公開サイトの「活動共有ノート」を更新する

## 記録の構成

- GitHubからの活動記録（自動更新）
- 専門用語のやさしい説明
- ジャーナリング（横塚翔太がNotionで追記する振り返り）

GitHubは履歴の材料であり、公開内容の正本はNotionに置く。秘密情報らしき文字列を含むコミットは、Notionへの自動記録から除外する。
