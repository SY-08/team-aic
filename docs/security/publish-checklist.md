# Publish Checklist（公開前チェックリスト）

TEAM AICは活動の過程をできるだけ公開する方針を取っていますが、公開してはいけない情報も明確に決めています。
GitHubへコミットする前・GitHub Pagesで公開する前に、必ず以下を確認してください。

## 公開NG項目

- [ ] APIキー・シークレットキーが含まれていないか
- [ ] アクセストークン・認証トークンが含まれていないか
- [ ] パスワード・パスフレーズが含まれていないか
- [ ] `.env` などの環境変数ファイルが含まれていないか
- [ ] 個人情報（住所・電話番号・生年月日など）が含まれていないか
- [ ] 家族や第三者が特定される情報が含まれていないか
- [ ] 守秘義務のある情報（取引先・案件の内部情報など）が含まれていないか
- [ ] 管理画面のURL・社内システムのURLが含まれていないか

## 確認の流れ

1. `git status` / `git diff` で変更内容を必ず目視確認する
2. 上記チェックリストに沿って、追加・変更したファイルを確認する
3. 該当する情報が見つかった場合は、コミット前に修正・削除する
4. 迷った場合は「公開しない」を優先する

## 補足

- Notion側で管理している内部情報は、公開用に整えた内容だけをGitHubへ中継します。
- スクリーンショットや画像を追加する場合も、上記項目が写り込んでいないか確認してください。
