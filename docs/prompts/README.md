# docs/prompts

Claude Codeなど、AIへの指示・プロンプトを蓄積するディレクトリです。

- 実際に使用した指示文とその結果
- うまくいかなかったプロンプトと改善後のバージョン
- 試行錯誤中の実験的なプロンプト

**公開前に必ず確認すること**：APIキー・トークン・パスワード・管理画面URLなどの機密情報が含まれていないか、
[`docs/security/publish-checklist.md`](../security/publish-checklist.md) に沿って確認してください。

まとまった内容は [`prompts.html`](../../prompts.html) にも反映していきます。
