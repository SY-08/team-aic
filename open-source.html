Open Source

公開方針

team AICは、設計図・実行ログ・Claude Codeへの指示・プロンプト・改善履歴を、できる範囲で公開していきます。

何を公開するか

「完成した成果」だけでなく「進めるための仕組み」そのものを公開する方針です。

🗺️ 設計図

サイトや仕組みの構成図・企画メモを Blueprints に置いていきます。

🪵 実行ログ

今日やったこと・決めたこと・次にやることを Live Build Log に記録します。

🤖 Claude Codeへの指示

Claude Codeに渡した指示やプロンプトを Prompts に蓄積します。

📈 改善履歴

試行錯誤や方針転換の記録も、隠さずに残していきます。

更新の仕組み

🧠
このサイトの内容は、Notionの「08 Website CMS（HP同期元・Notion正本）」を正本として管理しています。 今後、ページ内容を更新する場合はGitHub側を直接編集するのではなく、Notion側を編集し、自動同期の仕組みを通じてGitHubへ反映します。 仕組みの詳細はAutomationにまとめています。

公開しないもの

🔒
APIキー・トークン・パスワード・.env・個人情報・家族や第三者が特定される情報・守秘義務情報・管理画面URLは公開しません。 詳細は非公開の運用チェックリスト（docs/security/publish-checklist.md）に沿って確認しています。
