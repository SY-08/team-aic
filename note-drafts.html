Note Drafts

GitHub Pagesに蓄積した内容を、タイミングを見てnoteへコピペしていくための下書きコーナーです。

方針

team AICでは、まずGitHub Pages上に活動のログや考えを書き溜めます。 そのうえで、まとまりが出てきた内容や、より多くの人に届けたい内容を選び、タイミングを見てnoteへコピペして発信します。

GitHub Pages側は「進行中の一次情報」、noteは「読み物として整えた二次発信」という役割分担です。 そのため、このページの下書きは完成度よりも記録・整理を優先しています。

🔁
転記のタイミングは決め打ちにせず、内容がまとまった段階・伝えたいタイミングが来た段階で都度判断します。

下書き一覧

実際の下書き本文は docs/note-drafts/ に置いていきます。

方針

Instagram、X、YouTube、本化など、外へ出す前の文章や台本をここにまとめる。

完成記事ではなく、思考途中・実行途中の素材も置いてよい。
